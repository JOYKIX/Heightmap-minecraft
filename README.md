# Heightmap Minecraft Studio

Version allégée pour GitHub Pages : une base simple à maintenir avec seulement les fichiers nécessaires au site.

## Structure

```text
index.html   -> interface et wiring principal
style.css    -> styles UI consolidés
script.js    -> génération terrain + preview + export
```

## Démarrage

```bash
python3 -m http.server 8000
```

Puis ouvrir `http://localhost:8000`.
